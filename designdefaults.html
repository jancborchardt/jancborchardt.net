1. Introduction

Input fields



  • Clear label-to-field relations and as little noise as possible. Instead of having the labels above or next to the input field, the label should be inside the field whenever possible. This can be done with the placeholder attribute or using the InField Labels jQuery plugin. (check for browser compatibility, settle on either placeholder or infield labels)
  • Only one main scrollbar on the page. Textareas should have no scrollbars, but instead automatically resize vertically to fit the text content. Use the Autosize jQuery plugin.
  • Resize handle is obsolete. Remove them with CSS: textarea { resize:none; } (pull request to autosize)

Lightbox

Have a bunch of images display here for lightbox demo yea. Preface about how »Lightbox« is a shitty name.
  • Always display the whole image, as good as possible. The image should be displayed centered, either in original size or if bigger than the screen size, down-scaled to fit height or width, whatever touches edges first. Additionally it should have some margin for the image to not touch the edges.
  • Focus on the image, with the background subdued. The page should be overlayed with a grey half-transparent background with slight noise. The image should have a slight box-shadow and no border.
  • Make the relation to the photos on the page more obvious. Use an elastic effect like Fancybox.
  • Controls should always be in the same place. Placing the controls at the image or even on it causes them to move around as images and sizes change and make skipping images a pain. Instead, they should be absolutely positioned on the edges.
  • Make it easy to close the view. Getting out of the lightbox should be as easy as clicking anywhere outside the image, or specifically on the big × which should always be on the absolute top right.
